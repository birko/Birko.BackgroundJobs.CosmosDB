# Birko.BackgroundJobs.CosmosDB

Azure Cosmos DB persistent job queue for the Birko Framework.

## Features

- Persistent job queue with Cosmos DB storage
- Priority-based dequeue with scheduled job support
- Automatic retry with configurable max retries
- Job purge for completed/failed/cancelled jobs

## Dependencies

- Birko.BackgroundJobs
- Birko.Data.CosmosDB
- Microsoft.Azure.Cosmos
